# Smart Syllabus FAQ Chatbot

This project is a Smart Syllabus FAQ Chatbot built using Python, Streamlit, and Ollama.

## Technologies Used
- Python
- Streamlit
- Ollama (Llama3)
- Prompt Engineering

## How to Run the Project
1. Install requirements:
   pip install -r requirements.txt

2. Run Ollama:
   ollama run llama3

3. Run Streamlit app:
   streamlit run app.py

4. Open browser:
   http://localhost:8501
